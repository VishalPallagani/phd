# PhD Dissertation Website

This folder now contains only the static website for the dissertation and the files it actually uses.

## Main files

- `index.html`
- `styles.css`
- `script.js`
- `report_march25.pdf`
- `images/`

## Local use

Open `index.html` directly in a browser.

## GitHub Pages

1. Push this folder to a GitHub repository.
2. Open `Settings -> Pages`.
3. Choose `Deploy from a branch`.
4. Select your main branch and the repository root.
5. Save to publish the site.
